# Quick Start Guide

## For Office Users (Using the .exe)

### Step 1: Get the Application
- Ask your IT person for `GAC-PDF-EXCEL-CONVERTER.exe`
- Or download it from the shared network folder
- Save it to your Desktop or a convenient location

### Step 2: Launch
1. Double-click `GAC-PDF-EXCEL-CONVERTER.exe`
2. A window titled "Bank Statement PDF-to-Excel Converter v1.1" will open
3. The app is ready to use (no installation required!)

### Step 3: Convert Your Bank Statement

**Example Workflow:**

1. **Select PDF File**
   - Click "Browse..." next to "Select PDF File"
   - Navigate to your bank statement PDF
   - Click "Open"

2. **Check Preview**
   - Tables from the PDF appear in tabs below
   - Review the extracted data
   - Make sure everything looks correct

3. **Choose Sheets**
   - Check the boxes for sheets you want to keep
   - Uncheck any you want to exclude
   - (Usually keep everything checked)

3b. **Optional: Edit in-app preview (right-click menus)**
   - Right-click **sheet tabs** to rename, duplicate, insert, or delete a sheet
   - Right-click **row/column headers** to insert/delete rows/columns or sort a column
   - Drag **row/column headers** to reorder (the preview updates the sheet data)

4. **Set Output File**
   - Click "Browse..." next to "Save Excel As"
   - Choose where to save (Desktop is default)
   - Give it a filename like `March_2026_Statement.xlsx`
   - Click "Save"

5. **Convert**
   - Click the blue "Convert & Save" button
   - Wait a few seconds for processing
   - Status bar will show "✓ Excel file saved successfully"
   - Your Excel file is ready!

### Step 4: Use Your Excel File
- Open the saved Excel file in Microsoft Excel
- You'll see multiple sheets (tabs at bottom)
- Each sheet contains different data from the PDF
- Edit or delete rows/columns as needed
- Save normally in Excel

## Common Questions

**Q: Do I need Python installed?**  
A: No! The .exe includes everything needed.

**Q: Can I use my Mac?**  
A: Currently Windows only. Contact IT if you need Mac version.

**Q: What if I only want certain data?**  
A: You can edit the in-app preview first (rename/delete/insert/sort/reorder), or export to Excel and edit there.

**Q: Is my bank data safe?**  
A: Yes! Everything stays on your computer. No data goes to the internet.

**Q: Can I process multiple PDFs at once?**  
A: One at a time. Process each PDF separately.

**Q: The app looks different than the screenshots**  
A: Updates may have changed the layout. Core functionality is the same.

## Troubleshooting

**Problem: App won't start**
- Try running as Administrator (right-click → Run as administrator)
- Restart your computer
- Contact IT support

**Problem: PDF doesn't extract**
- Make sure the file is a real PDF (not renamed file)
- Try opening it in Adobe Reader first
- Some PDFs may not be compatible

**Problem: Excel file is blank**
- Make sure you selected at least one sheet (checkboxes)
- Try again with a different PDF to test
- Contact IT if problem persists

## Tips & Tricks

- Save your Excel files with dates in the filename (e.g., `Statement_March_2026.xlsx`)
- Before deleting data in Excel, make a backup copy
- You can edit the Excel file after creation - add notes, highlight important entries, etc.
- Share Excel files with colleagues via email or shared network folder
