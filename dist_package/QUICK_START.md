# Quick Start Guide

## For Office Users (Using the .exe)

### Step 1: Get the Application
- Ask your IT person for `GAC-PDF-EXCEL-CONVERTER.exe`
- Or download it from the shared network folder
- Save it to your Desktop or a convenient location

### Step 2: Launch
1. Double-click `GAC-PDF-EXCEL-CONVERTER.exe`
2. A window titled "Bank Statement PDF-to-Excel Converter v1.1" will open
3. The app is ready to use (no installation required!)

### Step 3: Convert Your Bank Statement

**Example Workflow:**

1. **Select PDF File**
   - Click "Browse..." next to "Select PDF File"
   - Navigate to your bank statement PDF
   - Click "Open"
   - The app will extract **all data** from the PDF automatically

2. **Review Preview**
   - All extracted data appears in the preview tab
   - Check that everything looks correct
   - **All PDF data is preserved** including wrapped text and multi-row entries

3. **(Optional) Split Data into Multiple Sheets**
   - If you want to divide the data into separate sheets, you can do this:
   - Click on the row where you want to split
   - Click "Mark split at selected row"
   - Click "Clear split markers" to remove all splits
   - *Don't worry if you don't split - you can always do it manually in Excel*

4. **Choose Sheet Name**
   - A checkbox appears for the Data sheet
   - Keep it checked to include it in the Excel file
   - (Typically you'll keep it checked)

5. **Set Output File**
   - Click "Browse..." next to "Save Excel As"
   - Choose where to save (Desktop is default)
   - Give it a filename like `March_2026_Statement.xlsx`
   - Click "Save"

6. **Convert**
   - Click the blue "Convert & Save" button
   - Wait a few seconds for processing
   - Status bar will show "✓ Excel file saved successfully"
   - Your Excel file is ready!

### Step 4: Use Your Excel File
- Open the saved Excel file in Microsoft Excel
- All data is in the "Data" sheet (or multiple sheets if you used splits)
- Review and edit as needed
- Everything from the PDF is preserved exactly as extracted

## Key Features
- ✅ **Extracts ALL data** from PDF - no data is skipped or filtered
- ✅ **Preserves exact layout** - headers, amounts, dates all in correct columns
- ✅ **Optional splitting** - divide data into multiple sheets before saving
- ✅ **In-app editing** - right-click to insert/delete rows/columns, sort, rename sheets, and drag to reorder
- ✅ **Simple workflow** - just select PDF, review, and save
- Save normally in Excel

## Troubleshooting

**Problem: App won't start**
- Try running as Administrator (right-click → Run as administrator)
- Restart your computer
- Contact IT support

**Problem: PDF doesn't extract**
- Make sure the file is a real PDF (not renamed file)
- Try opening it in Adobe Reader first
- Some PDFs may not be compatible

**Problem: Excel file is blank**
- Make sure you selected at least one sheet (checkboxes)
- Try again with a different PDF to test
- Contact IT if problem persists

## Tips & Tricks
